# Quest Academy - Gamified Learning App MVP

An MVP for a gamified mobile/web learning platform built using Flutter. Users can explore technical learning quests, track their experience points (XP), and earn visual badges.

## 🚀 Key Features Implemented
- **Multi-Screen Navigation:** Implemented dynamic constructors passing specific names and custom XP rewards cleanly between screen transitions.
- **Local Asset Integration:** Configured tracking for image folders and local verification audio tracks.
- **Modular Widgets:** Built clean reusable UI elements for achievement trackers and quest cards.

## 📱 App Flow Sequence
1. **Splash Screen:** Immersive initial app launch entry.
2. **Home Screen Dashboard:** Features an App Bar, a custom Navigation Drawer, sliding image/text Carousels, an Achievements ListView tracking badge milestones, and an available Quest GridView panel.
3. **Quest Detail Screen:** Extracts data constructor fields dynamically to show individual milestone layouts.
4. **Audio Verification Player:** Features a simulated audio listener trigger that redirects forward after validation.
5. **Reward Screen:** Displays responsive congratulations cards showing exact dynamic earned scores (+40 XP / +50 XP).
6. **Profile Screen:** Displays user statistics, overall scores, and settings.
